# Logistic Classifier

Implemented a binary classifier from scratch using:

- Sigmoid activation
- Binary Cross Entropy loss
- Gradient descent optimization

## Results:

- Accuracy: 86.7%
- Loss at Epoch 0  : 0.82
- Loss at Epoch 900 : 0.35

## Concepts Learned:
- Logistic regression
- Sigmoid function
- BCE loss
- Gradient descent
- Classification evaluation
